# Welcome to the GNOME Calendar application's issue tracker!

* To report a bug with this calendaring application, select the "Bug" template under "Description" above to provide the needed information in a structured way.

* To request a feature, please use the "Feature" template. Ideally, discuss with the [GNOME Design team](https://welcome.gnome.org/team/design/) or the GNOME Calendar development chatroom first.
